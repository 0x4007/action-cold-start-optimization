# Interactive Tutorials

Welcome to our interactive tutorial section! These tutorials will help you learn how to build WebAssembly-optimized GitHub Actions plugins step by step, with hands-on coding exercises and immediate feedback.

## Available Tutorials

### Beginner Tutorials

These tutorials cover the fundamentals of building plugins with our framework:

- [First Plugin Tutorial](/tutorials/beginner/first-plugin/) - Create your first GitHub Action plugin
- [Event Handling Tutorial](/tutorials/beginner/event-handling/) - Learn to handle multiple GitHub events
- [GitHub API Tutorial](/tutorials/beginner/github-api/) - Work with the GitHub API

### How the Interactive Tutorials Work

Our interactive tutorials feature:

1. **Live Code Editor** - Edit code directly in your browser
2. **Step-by-Step Instructions** - Clear guidance for each part of the process
3. **Immediate Execution** - Run your code to see the results instantly
4. **Performance Metrics** - View real-time performance statistics

## Getting Started

To begin, select one of the tutorials above. Each tutorial will guide you through the process of building a functional GitHub Action plugin. You'll learn both the fundamentals and best practices for optimizing your plugins with WebAssembly.
